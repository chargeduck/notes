2. tooltip 提示 ​

提示框组件，用于配值鼠标滑过或者点击图标时的显示框

  1. trigger 触发类型
触发类型描述
item鼠标滑过图标时触发
axis鼠标滑过坐标轴时触发
  1. triggerOn 触发时机

鼠标滑过或者点击图标时触发

触发类型描述
mouseover鼠标移过的时候展示
click鼠标点击的时候展示
  1. formatter 格式化函数

用于自定义提示框的内容

js
const option = {
  tooltip: {
    trigger: 'axis',
    axisPointer: {
      type: 'line'
    },
    triggerOn: 'click',
    formatter: function (args) {
      // 具体参数可以从args中获取
      console.log(args)
      // 格式化提示框的内容
      return `${ args[0].axisValue } 周阅读人数: ${ args[0].value }`;
    }
  }
}

const option = {

    type: 'scatter',

    symbolSize (params) {
      const bmi = params[1] / params[0] / params[0]
      return bmi > 18 ? 20 : 10
    }
  }]

  1. 气泡不同颜色

给series设置itemStyle属性即可，通过回调函数可以动态设置气泡颜色

js
const option = {
  series: [{
    name: '散点图',
    data: [[100, 200], [200, 100], [300, 400], [180, 200]],
    type: 'scatter',
    // 气泡效果
    symbolSize (params) {
      const bmi = params[1] / params[0] / params[0]
      return bmi > 18 ? 20 : 10
    },
    // 不同颜色
    itemStyle: {
      color (params) {
        const bmi = 10000 * params.data[1] / params.data[0] / params.data[0]
        return bmi > 24 ? 'red' : 'green'
      }
    }
  }]
}

  /**
   * 1. 显示基本的地图
   * 2. 把数据设置给series
